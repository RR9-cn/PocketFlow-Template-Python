# 【基础篇】如何构建一个好故事

## 前言：我怎么跟AI玩起来

我用AI写东西，发现AI像个特别热情，但有点缺心眼的实习生。

它交上来的稿子，永远格式完美，用词标准，但读起来总觉得缺点人气儿，像一杯忘记放糖的奶茶，该有的都有，就是不甜。

我花了好一阵子才琢磨明白：**AI不是作家，它是个捧哏。** 一个记忆力超群、啥梗都接，但完全没谈过恋爱、没挤过早高峰的捧哏。

你给它一个无聊的话题，它能陪你一本正经地聊八百回合。但你要是想让它逗乐，你就得先当那个说相声的，把包袱递给它。

这本手册，就是我的"逗哏"指南。它告诉你，怎么带着你家这位天才捧哏，一起说一段好玩的相声，而不是被它带进"尊敬的用户，我很高兴为您服务"的死胡同里。

忘掉那些复杂的理论。记住一句话：

**作品的可爱度 = AI的热情 x (你给的梗 + 你逗它的水平)**

AI的热情（大模型）只会越来越高。你要做的，就是想好玩的梗，和学会怎么逗它。

整个流程，分为四步：**捡饼干，泡壶茶，聊聊天，做个SPA。**

我们从第一步开始吧。

---

## 第一章：捡饼干 —— 当你不知道晚饭吃什么时

你想写，但脑子里空空如也，像刚洗过的碗。

这是好事。空的碗才能装好吃的。

别自己硬想。人的思路总爱在熟悉的小区里遛弯，一不小心就绕回原点。我们要的，是出门撞见邻居家的猫，那种小小的意外。

所以，我们用这个工具：**"灵感饼干罐"**。

它的原理很简单：用一些可爱的随机组合，把你从"写个宏大故事"的牛角尖里拽出来。

"饼干罐"里有三类小饼干：

*   **身份饼干：** 故事里有谁？
*   **场景饼干：** 他们在哪儿？
*   **事件饼干：** 他们在干嘛？

**执行方法：** 打开卷末的《创世关键词库》。想象你面前有个饼干罐，闭上眼伸手进去摸。摸出3-5个"身份"，3-5个"场景"，3-5个"事件"。凑够10个左右就行。

比如，你摸出了这样一把：

**示例A (温情向)**：
*   **身份 (3):** 退休的魔法少女、社区保安大叔、一只总是睡不醒的橘猫
*   **场景 (3):** 24小时便利店、楼顶天台、凌晨三点的厨房
*   **事件/概念 (4):** 寻找丢失的拖鞋、尝试种一盆多肉、熬夜看一部老电影、完美地煎一个荷包蛋

**示例B (爽文向)**：
*   **身份 (3):** 咸鱼、维护局精英、门卫大爷
*   **场景 (3):** 拉面馆、世界边境、虚空
*   **事件/概念 (4):** 拯救世界、开业大酬宾、修复世界、吃火锅

看到这些词，是不是觉得有点好笑？它们不是史诗，它们是薯片和可乐。是搭一个舒服小窝用的最舒服的材料。

你现在有了一堆温暖的、看似不搭界的乐高积木。

很好。捡饼干阶段完成。

接下来，你要用第二章的方法，给这些积木注入灵魂，让它们活过来，会呼吸，还会打哈欠。

---

## 第二章：泡壶茶 —— 你只需要回答六个温暖的问题

动笔之前，别碰AI。看着你摸出来的那堆"饼干"，给自己泡壶好茶，回答我六个问题。

这六个问题，就是你故事的"人设与关系卡"。是你家AI捧哏需要了解的"包袱核心"和"情感基调"。这套方法，我叫它**"角色茶话会"**。

我们用两个例子来演示，看看同样的方法，怎么既能写温情小品，也能写咸鱼爽文。

---

### 示例A：温情治愈向 - 退休魔法少女

我们决定让"退休的魔法少女"当主角。现在，开始聊天：

1.  **她是谁？（身份）**
    *   *不是简单重复。而是用饼干组合出她的核心萌点。*
    *   **示例：** 她是曾经拯救过世界的魔法少女，如今退休了，住在老城区，最大的法器"星辰之杖"现在用来当晾衣杆。她想过普通人的生活，但偶尔还是会不自觉地用魔法，比如用"火焰咒"点蚊香。

2.  **她和谁有关系？（情感纽带）**
    *   *这是故事的温度计。人物不是孤岛。*
    *   **示例：** 她和总在小区门口值班的保安大叔很熟。大叔有点爱唠叨，总提醒她按时吃饭，觉得她一个年轻姑娘家家的生活太不规律。还有一只橘猫，是她和大叔一起喂养的"小区公务员"。

3.  **她想要什么？（欲望）**
    *   *越小、越具体、越生活化越好。*
    *   **示例：** 她现在什么都不想拯救，就想在楼顶天台上，成功地种活一盆多肉植物。

4.  **为什么想要？（动机）**
    *   *这决定了故事的深度和温度。*
    *   **示例：** 因为保安大叔送了她这盆多肉，说是"年轻人养点东西，生活才有奔头"。她不想辜负大叔的好意，也想证明自己除了拯救世界，也能照顾好一个小生命。这是一种笨拙的回应和在乎。

5.  **是什么在阻碍她？（障碍/笑点）**
    *   *冲突不一定是坏人，也可以是可爱的麻烦。*
    *   **示例：** 她对植物一窍不通。想用"甘霖术"浇水，结果水太多，差点淹死多肉；想用"日光术"补充光照，结果把叶子烤焦了一点。那只橘猫还总想去刨盆里的土。

6.  **如果失败了会怎样？（后果）**
    *   *后果不必是世界末日，可以只是有点小尴尬和温柔的延续。*
    *   **示例：** 如果多肉死了，她会有点难过，觉得对不起大叔。她可能会偷偷再去买一盆一模一样的，假装自己养得很好。而大叔可能早就看穿了，第二天又会提着一盆新的、更好养的仙人球出现在她面前。

---

### 示例B：咸鱼爽文向 - 拉面馆老板

我们决定让"咸鱼拉面馆老板"当主角。同样六个问题：

1.  **他是谁？（身份）**
    *   **示例：** 他叫陈舟，曾经是维护局最强的研究员，三年前因为理论"太超前"被开除。现在每天躺平吃泡面，最大的愿望是睡到自然醒。他有能修复世界的能力，但只想开拉面馆。

2.  **他和谁有关系？（情感纽带）**
    *   **示例：** 和维护局的季思思（精英但总被他怼）、古德拜副局长（老熟人）、门卫老王（神秘大爷）有联系。关系基调是"一碗面的交情"——在乎但不黏糊，帮忙但要收钱。

3.  **他想要什么？（欲望）**
    *   **示例：** 他只想好好开他的拉面馆，每天研究是推出大宽还是二细，别被打扰。

4.  **为什么想要？（动机）**
    *   **示例：** 因为三年前拯救世界累得要死，结果被当疯子踢出局。他受够了伟光正的使命感，只想过普通人的生活。拉面馆是他证明"不拯救世界也能活得很好"的方式。

5.  **是什么在阻碍他？（障碍/笑点）**
    *   **示例：** 世界总是要完蛋。元素紊乱、虚空掠食者、各种破事儿。维护局的人老来求他。季思思的教科书理论总是出错，需要他擦屁股。还有，他的外卖总是因为世界危机送不到。

6.  **如果失败了会怎样？（后果）**
    *   **示例：** 如果他不出手，世界真的会完蛋，他的拉面馆也开不成。但他绝不会为了"正义"出手，只会为了"我的黄焖鸡没了"这种理由。

---

**看出区别了吗？**

**同样的六个问题**，但：
- 温情向：欲望小（种多肉）、障碍可爱（魔法失控）、后果温柔（大叔再送一盆）
- 爽文向：欲望反差大（能救世界但只想开店）、障碍是打脸素材（精英总出错）、后果是装逼机会（不救世界拉面馆也开不成）

**方法是一样的，只是填入的内容不同。**

现在，这张充满了人情味的人设卡有了。你可以去找你那个傻乎乎的捧哏朋友了。

---

## 第三章：聊聊天 —— 怎么跟AI愉快地对话

你不能对AI说："根据这些设定，写个温馨搞笑的故事。"

它会给你一堆"她的心中涌起一股暖流"、"空气中弥漫着尴尬又好笑的气氛"这种正确的废话。

你得像约朋友出门玩一样，先定好去哪儿、玩什么，并说好"咱们今天不吵架"。

**第一步：角色扮演 (Persona Pattern)**
先给AI一个身份。这决定了它的"聊天风格"。

**温情向指令**：
"忘掉你是AI。你现在是一个脑洞很大、喜欢写生活段子的网络写手，说话要有趣、接地气、充满人情味。"

**爽文向指令**：
"忘掉你是AI。你现在是一个专写爽文的网文作者，擅长设计打脸桥段、装逼金句，节奏要快，笑点要密，让人停不下来。"

**第二步：分享人设卡 (Context Injection)**
把你第二章写的"角色茶话会"的答案喂给它。

**指令格式**：
"接下来，咱们一起写个故事。这是主角的人设与关系卡，你记一下：
【主角核心档案】
*   **身份：** [第1个问题的答案]
*   **情感纽带：** [第2个问题的答案]
*   **欲望：** [第3个问题的答案]
*   **动机：** [第4个问题的答案]
*   **障碍：** [第5个问题的答案]
*   **失败后果：** [第6个问题的答案]"

**第三步：划好聊天边界 (Negative Prompt)**
你必须明确告诉AI，哪些是"禁区"。

**通用禁令** (温情和爽文都要遵守):
```
写作时，咱们必须遵守几个核心规则：

【禁止】冰冷的交易关系
人物之间的互动要有温度，哪怕是拌嘴和争执，也要能看出他们彼此在乎，有感情基础。他们不是推动剧情的工具人。

【禁止】大喊大叫
角色可以有激烈的情绪，但绝不能通过"大吼"、"咆哮"、"怒斥"来表达。请用紧绷的沉默、一个用力的关门动作、一句压低声音的话、或一个欲言又止的眼神来代替。要保持人物应有的体面和克制。

【禁止】廉价的比喻和抒情
尽量别用'像'、'仿佛'、'如同'。也别用'一丝'、'一抹'、'深邃'、'缓缓地'、'精致'、'完美'这些词。咱们说人话，讲实事。

【必须】用行为说话
绝对不要直接写人物的心理活动和情绪。不许写'她感到好笑又无奈'。请改成'她看着被水淹的多肉，叹了口气，然后用手指戳了戳橘猫的脑门'。用角色的动作、对话和细节来展示他的状态。
```

这套组合拳打出去，AI生成的文本"实习生味"会减少90%。它从一个写标准公文的机器，变成了一个懂分寸、知冷暖的聊天搭子。

它给出的，是"草稿"。接下来，轮到你给故事做个舒服的SPA了。

---

## 第四章：做个SPA —— 拿起你的按摩油和修容刀

AI给你的草稿，你不能直接用。你需要做四件事，让它从"还行"变成"可爱到心坎里"。

我称之为**"后期护理四步法"**：**去角质、换活血、小碎步、加点香薰。**

**第一步：去角质（删）**
拿到文本，第一件事就是删掉多余的"死皮"。
*   **删掉所有形容词和副词。**
    *   AI写："她小心翼翼地拿起那个圆滚滚的鸡蛋。"
    *   你改成："她拿起鸡蛋。"
*   **删掉所有解释。**
    *   AI写："因为她想快点吃到，所以她把火开大了。"
    *   你改成："她把火开大了。" 相信读者能懂她的着急。
*   **删掉所有啰嗦的过渡。** "与此同时"、"另一方面"、"值得一提的是"，看到就删。

**第二步：换活血（换）**
*   **用有趣的动词，换掉平淡的动词。**
    *   "她慢慢地走" -> "她蹭过去" 或 "她踮着脚尖溜达"。
    *   "保安大叔说" -> "保安大叔清了清嗓子，背着手开口"。
*   **用克制的动作，换掉激烈的情绪描写。**
    *   AI写："他愤怒地对她大喊。"
    *   你改成："他嘴唇动了动，最后什么也没说，只是把头转向了一边。" 或者 "他把手里的报纸捏紧了，发出轻微的声响。"
*   **用好玩的对话，换掉内心独白。**
    *   AI写："她心想，这下完蛋了，今天的早餐又泡汤了。"
    *   你改成：她对着那只橘猫说："看见没，这就是魔法的代价。今天咱俩都喝西北风。"

**第三步：小碎步（拆）**
AI喜欢写工整的长句子。但生活中的念头和动作，都是碎片化的。
*   **把长句拆成短句。**
    *   AI写："当她手忙脚乱地试图用一个水系魔法扑灭灶台上的小火苗时，那只橘猫趁机跳上桌子，用它的小爪子拨动着盘子里剩下的那个完好无损的鸡蛋。"
    *   你改成："灶台着火了。她赶紧念咒。一个水球过去。火灭了。她回头。猫上桌了。爪子正搭在最后一个鸡蛋上。"
*   **用短句制造轻松的节奏感。** 就像动漫里的搞笑桥段，镜头切换很快。

**第四步：加点香薰（加）**
前面都是做减法，最后一步，做加法。但只加三样能升华情感的东西。
*   **加AI忽略的感官。** AI是视觉动物。你要补上听觉、嗅觉、触觉。
    *   在安静的厨房里，加上"冰箱压缩机嗡的一声启动了，吓了她一跳。"
    *   在对峙的场景里，加上"他闻到了对方身上廉价的烟草味，混着雨水的湿气。"
    *   在角色紧张时，加上"他感觉到汗水粘住了衬衫的后背，又冷又痒。"
*   **加"可爱的瑕疵"。**
    *   在对话里，加上一点点口头禅、小小的结巴，或者自言自语。
    *   "我……我寻思着，今天这个……多肉，肯定能行。哎呀！"
    *   这才是活生生的人。AI是完美的，但可爱的人不是。
*   **加"情感的粘合剂"。**
    *   这是最重要的。在人物互动中，加入微小的、非语言的善意。一个共享的眼神，一次无言的拍肩，一次心照不宣的沉默，递过来的一杯热水，或者是在对方出糗时忍住不笑的努力。这些细节是滋养人物关系的土壤。

做完这四步，你会发现，那段有点呆板的AI文本，开始有了心跳和体温。

## 【基础篇】结语

这套方法，很简单。

*   **灵感饼干罐，** 找到好玩的起点。
*   **角色茶话会，** 泡出温暖的羁绊。
*   **愉快聊聊天，** 带上你懂分寸的AI搭子。
*   **后期做SPA，** 亲手注入生活的温度。

AI是个好玩的搭子。它能帮你快速想出一些无厘头的点子，填充你懒得写的场景。它不知疲倦，随时陪你聊天。

但它永远给不了你三样东西：

1.  **你自己的生活体验。** 你被窝里的温度，偷吃冰箱里最后一块蛋糕的心虚，与在乎的人拌嘴后又后悔的瞬间。这些是你的，不是它的。
2.  **你独特的幽默感与分寸感。** 你知道哪句吐槽最精准，哪个笑点刚刚好，什么时候的沉默比千言万语更有力。
3.  **你独有的温柔。** 故事有无数种可能，你选择最让你感到被世界温柔以待的那一种。

最终，让一个故事温暖可爱的，不是文字，而是文字背后那个愿意发现生活乐趣、珍视人与人之间联结的你。

而这一切，都藏在你敲击键盘的指尖里。

**到这里，你已经能写出有血有肉的故事了。**

如果你想让故事**更抓人、更让人停不下来**，继续往下看【提高篇】。

如果你只想写个温馨小品，恭喜你，基础篇已经够用了。去和你的AI搭子，讲个温暖又好玩的故事吧。

# 【提高篇】如何让故事更抓人

> 前置要求：你已经学会【基础篇】的四步法
> 适用场景：爽文、网文、需要"停不下来"感的故事

---

## 核心理念

**基础篇**教你"怎么讲一个好故事" (人物、关系、情感)
**提高篇**教你"怎么让读者停不下来" (爽点、节奏、高潮)

**两者关系**：
- 基础是骨架，提高是肌肉
- 先有基础的人物，再加提高的爽点
- 基础决定故事能不能站稳，提高决定能不能跑得快

---

## 🎯 爽点强度分级标准

爽点不是越多越好，而是要**分级、分布、有节奏**。

| 等级 | 符号 | 读者反应 | 出现频率 | 典型场景 |
|------|------|----------|----------|----------|
| **S级** | ⭐⭐⭐⭐⭐ | "卧槽！太爽了！" | 1-2次/全文 | 主角展现真实实力、碾压Boss、身份大反转 |
| **A级** | ⭐⭐⭐⭐ | "哈哈，太解气了！" | 3-5次/全文 | 打脸关键角色、获得重大奖励、预言成真 |
| **B级** | ⭐⭐⭐ | "有意思" | 8-12次/全文 | 小打脸、装逼成功、巧妙化解危机 |
| **C级** | ⭐⭐ | "还不错" | 15-20次/全文 | 幽默对话、日常装逼、小优越感 |

**记住**：
- C级是"底味"，让读者愿意继续读
- B级是"调味"，让读者觉得有意思
- A级是"提神"，让读者精神一振
- S级是"高潮"，让读者忍不住拍桌子

---

## 📊 四幕结构与爽点分布

**不管你写什么类型，爽点分布都遵循这个规律**：

```
开局要密集 → 中期要持续 → 过渡要埋伏笔 → 结局要爆发
```

### 🎬 第一幕：开局 (0-20%)

**目标**：在前500字抓住读者

| 位置 | 强度 | 怎么做 | 示例 (咸鱼文) | 示例 (温情文) |
|------|------|--------|--------------|--------------|
| 开篇30-100字 | C级 | 荒诞钩子/打破常规 | 水龙头喷果冻 | 魔法少女的法杖当晾衣杆 |
| 200-300字 | C级 | 人设反差 | "我对这些不感兴趣" | 用火焰咒点蚊香 |
| 400-600字 | B级 | 小打脸/笑点 | 马桶被当成上古神器 | 浇水术把多肉淹死 |
| 800-1000字 | A级 | 预言成真 | 法阵炸出牛奶湖 | 橘猫真的把多肉刨了 |

**AI写作指令**：
```
第一幕写作要求 (前1000字):
1. 开篇用[具体的荒诞事件]作为钩子，不要写"奇怪的事情发生了"这种废话
2. 在前100字内，用一个行为展示主角的[特点]，不要用心理描写
3. 在300字处，安排配角登场，通过对话展现主角和配角的反差
4. 在800字处，让主角的某个判断/预言立即得到验证，打脸配角
5. 节奏：每200字必须有1个笑点或让人会心一笑的细节
```

---

### 🎬 第二幕：升级 (20%-60%)

**目标**：维持热度，推向第一个大高潮

| 阶段 | 位置 | 强度 | 怎么做 |
|------|------|------|--------|
| 铺垫期 | 20%-30% | C-B级 | 日常装逼，展示能力冰山一角 |
| 蓄力期 | 30%-40% | B-A级 | 解决小危机，引出更大威胁 |
| **第一高潮** | **40%-50%** | **S级** | **主角实力全开，震撼所有人** |
| 缓冲期 | 50%-60% | C-B级 | 收尾，回归日常，给读者缓口气 |

**第一高潮 (S级) 标准结构** (400-600字):
```
【铺垫】(100-200字)
- 所有人认为主角会失败
- 营造绝望氛围
- 至少3个配角说"不可能"

【爆发】(200-400字)
- 主角展现真实实力
- 必须包含3个连续爽点：
  ① 视觉冲击：华丽的动作/技能描写
  ② 理念碾压："我不是来遵守规则的"这类金句
  ③ 反差萌：用日常比喻描述神迹 (如"像揉面团一样")

【收尾】(100-200字)
- 配角震惊反应
- 主角依然淡定 (甚至打哈欠/想着吃饭)
```

**示例对比**：

**咸鱼爽文** (跳入虚空修复世界):
```
【铺垫】"完了，这次真的没救了"
【爆发】纵身一跃 → 编织虚空之网 → "像和面一样"
【金句】"我不是来遵守理论的"
【收尾】众人目瞪口呆，主角："累了，回去睡觉"
```

**温情治愈** (魔法少女终于养活多肉):
```
【铺垫】多肉已经蔫了，大叔说"算了，我再送你一盆"
【爆发】她突然想通了 → 不用魔法 → 查资料 → 正确浇水
【细节】她蹲在天台，一滴一滴地浇，橘猫在旁边陪着
【收尾】第二天多肉真的精神了，大叔看到后欣慰地笑了
```

**看出来了吗？同样是高潮，但表现形式完全不同**：
- 爽文：震撼、碾压、装逼
- 治愈：温柔、坚持、小确幸

**AI写作指令**：
```
第二幕第一高潮写作要求 (400-600字):
1. 用200字铺垫绝望氛围，让读者觉得"这次真的完了"
2. 主角出手分为3步：
   - 步骤1 (50字): 一个简单动作引发震撼
   - 步骤2 (150字): 具体技能/行为描写，必须用日常事物做比喻
   - 步骤3 (50字): 一句装逼金句或温柔细节
3. 收尾必须有配角反应 + 主角淡定的反差
4. 如果是温情向：把"震撼"换成"感动"，把"装逼"换成"温柔细节"
```

---

### 🎬 第三幕：过渡 (60%-80%)

**目标**：避免审美疲劳，埋伏笔，为第二高潮蓄力

| 位置 | 强度 | 怎么做 | 作用 |
|------|------|--------|------|
| 60%-65% | B级 | 身份揭示 | "原来你是XX" |
| 65%-70% | C级 | 日常装逼 | 喝茶/吃饭时顺手解决小事 |
| 70%-75% | B级 | 伏笔回收 | 第一幕埋的梗被揭开 |
| 75%-80% | A级 | 危机预警 | 引出终极Boss/危机 |

**注意**：
- ⚠️ 这是最容易让读者弃文的部分
- ✅ 必须每500字有1个B级以上爽点
- ✅ 可以加入情感线/幽默桥段调剂

**AI写作指令**：
```
第三幕日常过渡写作要求:
1. 安排2-3个日常场景 (开店/吃饭/聊天)
2. 每个场景模式: 意外事件 → 主角轻松化解 → 旁观者震惊
3. 在70%位置揭示一个伏笔 (如主角的过去/某个道具的真实用途)
4. 在78%位置让神秘角色出现，预示终极危机
5. 节奏: 每300字一个笑点，每600字一个爽点
```

---

### 🎬 第四幕：终极高潮 (80%-100%)

**目标**：用最强爆发点结束全文，让读者意犹未尽

| 阶段 | 位置 | 强度 | 怎么做 |
|------|------|------|--------|
| 危机降临 | 80%-85% | B级 | Boss登场，展示恐怖实力 |
| **第二高潮** | **85%-93%** | **S级** | **碾压式胜利** |
| 温馨结尾 | 93%-98% | C-B级 | 回归日常，呼应开局 |
| 彩蛋 | 98%-100% | C级 | 暗示续集/留悬念 (可选) |

**第二高潮必须比第一高潮更强**，三个方法：
1. **规模更大**：第一次救一个人 → 第二次救全世界
2. **时间更短**：第一次10分钟 → 第二次10秒
3. **态度更拽**：第一次认真 → 第二次吃着烧烤顺手解决

**标准结构** (600-1000字):
```
【危机】(100字): Boss强到离谱，所有人绝望
【反转1】(50字): 主角出现，众人看到希望
【装逼】(100字): 主角轻蔑举动 (如"你瞅啥?")
【反转2】(50字): Boss怒攻
【爆发】(300字): 主角秒杀，用夸张比喻描写
【金句】(20字): "就这?" / "我还没用力"
【反应】(80字): 众人震撼 + 敌人认怂
【收尾】(100字): 主角回归日常，呼应开局
```

**AI写作指令**：
```
第四幕终极高潮写作要求:
1. Boss出场用200字渲染恐怖 (击败了XX强者/毁灭了XX)
2. 主角出场第一句话必须是装逼金句
3. 战斗分3步:
   - 主角挑衅 (50字)
   - Boss怒攻 (100字)
   - 主角秒杀 (300字，必须用夸张比喻如"像拍苍蝇")
4. 战后必须有配角震惊独白
5. 最后100字主角回归日常，呼应开局场景
6. 如果是温情向：把"秒杀"换成"温柔的坚持"，把"装逼"换成"不动声色的守护"
```

## 🎨 爽点类型工具箱

这是写作手法库，根据需要随时取用。

### 1. 打脸爽 (最通用)

**公式**: 对方轻视 → 主角打脸 → 对方震惊

**变种**:
- **知识打脸**: 专家的理论被实践推翻 (季思思的法阵)
- **实力打脸**: "废物"秒杀天才
- **预言打脸**: "你会后悔"立即成真
- **身份打脸**: "乞丐"原来是首富

**模板**:
```
【轻视】"就你？连我一招都接不住"
【打脸】主角一指点出，对方倒飞
【震惊】"这...这是XX神功！你怎么可能会！"
```

**温情版打脸**：
```
【轻视】大叔: "养多肉很难的，你肯定养不活"
【打脸】一个月后多肉开花了
【震惊】大叔: "你...你是怎么做到的？"
```

---

### 2. 装逼爽

**公式**: 看似平常的行为 → 实际牛逼效果 → 众人震惊

**技巧**:
- **工具反差**: 用破铜烂铁做出神器效果
- **态度反差**: 打哈欠时顺手秒杀Boss
- **比喻反差**: "就像切菜一样"斩杀恶龙

**金句库**:
- "不好意思，我没用全力"
- "刚才是热身"
- "还有谁？"
- "就这？"
- "一般一般，世界第三"

**温情版装逼**:
- "这个啊，我只是碰巧知道一点"
- "也没什么，就是多试了几次"
- "其实挺简单的，你也可以"

---

### 3. 咸鱼爽 (佛系主角专用)

**公式**: 主角拒绝 → 事态恶化 → 主角勉强出手 → 轻松解决

**核心要素**:
- 动机接地气 (为了吃饭/睡觉/不被打扰)
- 出手过程轻松 (打哈欠/吃东西时顺手)
- 收尾继续咸鱼 (解决完继续躺平)

**示例**:
```
"别烦我，我要吃火锅"
→ 世界即将毁灭
→ "真麻烦，一分钟解决"
→ 打响指，世界恢复
→ "行了，火锅店要关门了，快走"
```

---

### 4. 系统流爽点

**核心爽点**:
- 获得逆天奖励
- 系统震惊/被怼
- 完成不可能任务
- 触发隐藏成就

**模板**:
```
【系统】"叮！获得新手奖励[小还丹]"
【主角】"就这？垃圾系统"
【系统】"！！！(此物可让人起死回生！)"
【主角】"哦，那还行"
【系统】"........"
```

---

### 5. 荒诞幽默爽

**方法**: 严肃场合 + 不协调元素

**示例**:
- 拯救世界的条件是"一碗拉面"
- 战场上摆烧烤摊
- 用拖鞋击败Boss
- 魔法世界讨论外卖速度

## 【提高篇】结语

记住核心公式:
- **基础篇**给你骨架 (人物、关系、情感)
- **提高篇**给你肌肉 (爽点、节奏、高潮)

**使用顺序**:
1. 先用基础篇的四步法塑造人物
2. 再用提高篇的四幕结构安排爽点
3. 最后用SPA四步法精修文本

**记住**：
- 温情文也可以有爽点 (只是表现形式更温柔)
- 爽文也必须有人物 (否则只是爆米花，吃完就忘)
- 最好的故事 = 真实的人物 + 精彩的节奏

# 【通用写作规则】

## 标题生成公式

**公式一：身份/时间反差 + 关系/行为突变**
- 示例: "离婚当天，前夫跪求复合"
- 示例: "退休魔法少女，只想种多肉"

**公式二：平淡开局 + 意外转折**
- 示例: "我只是来修马桶的，没想到..."
- 示例: "咸鱼拯救世界？求你们了，让我躺平吧"

**公式三：暧昧情景 + 合理借口**
- 示例: "半夜接到她的电话，说锁坏了"

- 标题15字以内

- 必须有吸引力
